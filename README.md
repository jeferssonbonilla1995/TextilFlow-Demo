---

## 🎯 ¿Qué es TextilFlow?

**El problema:** Empresas textiles manejando fichas FÍSICAS de papel por cada área. Se manchan, se pierden, no hay reportes confiables.

**La solución:** Digitalización completa con trazabilidad en tiempo real, scanner de unidades, integración con Microplus y cálculo de eficiencia (SAM).

**✅ YA funcionando en una empresa textil real en Ecuador.**

---

## 🏭 Áreas del Sistema

| Área | Función |
|------|---------|
| 🎨 **Diseño** | Creación del proyecto digital / Especificaciones de prenda |
| ✂️ **Corte** | Integración con Microplus para insumos / Tallas y colores |
| 🧵 **Bordado** | Estado de proceso / Seguimiento de unidades |
| 🖨️ **Impresión Digital** | Control de cantidades por talla/color |
| 👕 **Confección** | Registro de completado / Detección de fallas |
| 📦 **Bodega Insumos** | Búsqueda de insumos en Microplus / Costos |
| ✅ **Bodega Terminado** | Verificación con scanner / Cuadre final |

---

## 📷 Sistema de Scanner (Funcionalidad Clave)

**No es solo software. Es software + hardware trabajando juntos.**

1. El sistema indica: "Debes tener 100 prendas talla M, color rojo"
2. Operario pasa **prenda por prenda** por el scanner
3. El sistema cuenta automáticamente
4. Detecta si faltan o sobran unidades
5. Bodega terminado verifica con scanner otra vez

**Si falta una prenda o hay fallas, el sistema lo marca y no permite cerrar la ficha.**
